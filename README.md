# 📖 Dictionary App

An interactive dictionary application built with **React.js** that provides word definitions, phonetics, synonyms, examples, and visual context through images.

**Live demo:** [ritas-dictionary.netlify.app](https://ritas-dictionary.netlify.app/)

---

## 📋 About

This project was built as part of the **SheCodes React** bootcamp. It combines two separate APIs (a dictionary API and an image API) to create a rich, engaging experience when looking up words.

---

## ✨ Features

- 🔍 Search any English word
- 📝 Definitions with part of speech
- 🔁 Synonyms for each word
- 💬 Example sentences in context
- 🖼️ Relevant images pulled from an image API
- 🧹 Clean, minimal interface

---

## 🛠️ Built With

| Technology | Purpose |
|------------|---------|
| React.js | UI framework |
| React Hooks (useState, useEffect) | State and side-effect management |
| Dictionary API | Word definitions and synonyms |
| Image API | Visual context for searched words |
| CSS | Styling and layout |
| Netlify | Deployment |

---

## 📚 What I Learned

- How to integrate **multiple APIs** in a single React application
- How to manage multiple pieces of state that depend on the same user input
- How to render **conditional content** — only showing synonyms, examples or images when available
- How to handle API responses that have nested or variable data structures
- How to keep components clean and focused on a single responsibility
